To create the plume-util jar in libs/plume-util-1.1.5.jar, do the
following:

Build this version of the Checker Framework (the one with this file) or
the nondet-checker-experimental-flag branch

Run these commands:

git clone https://github.com/t-rasmud/plume-util.git
cd plume-util
git checkout run-det-checker
./gradlew jar -PcfLocal
# Assumes Checker Framework located in the $CHECKERFRAMEWORK directory
cp build/libs/plume-util-1.1.5.jar $CHECKERFRAMEWORK/libs/plume-util-1.1.5.jar
